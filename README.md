```text

│

├── 张艺兴 - 一直都在.mp3    # 全局背景音乐 (请确保文件名对应)
├── 张艺兴 - Would U Be Mine.mp3  # 黑胶唱片音乐
│
└── img/                   # 【重要】所有照片必须放在这个文件夹里！
    ├── MovieCover_mumu.jpg
    ├── PsychicCover.jpeg
    ├── 20250530_Beijing.jpg
    ├── WouldUBeMine.jpg
    └── ... (你的其他照片)
```
## 🛠️ 2. 如何修改成你的内容？
你可以直接在 GitHub 网页上点击 index.html 旁边的 ✏️ (编辑) 按钮，或者在本地用记事本打开修改。

1️⃣ 替换照片
将你的照片上传到 img 文件夹。 在代码中搜索 .jpg 或 .png，找到类似 ```<img src="img/MovieCover_mumu.jpg">``` 的代码，将文件名改为你上传的照片文件名。

2️⃣ 修改文字
在代码中搜索汉字（例如搜索“初见小马”），直接修改为你自己的心情记录。

搜索 Solar：替换成你的昵称。

搜索 2025年 4月：替换成你的入坑或看演出的时间。

3️⃣ 修改日历高亮
搜索代码 data-event。

```<span class="day-cell highlight-day">```：表示这一天会发光（有行程）。

如果某天没有行程，删除 ```highlight-day``` 和后面的 ```<span class="event-label">...</span>``` 即可。

4️⃣ 修改音乐
将你喜欢的 MP3 文件上传到仓库根目录。 找到代码最顶部的 ```<audio> ```标签，修改 ```src="..." ```中的文件名为你的音乐文件名。

## 🚀 3. 如何发布给朋友看？ (GitHub Pages)
你可以利用 GitHub 免费生成一个网页链接，发给朋友们在手机上打开。

在你的 GitHub 仓库页面，点击上方的 Settings (设置)。

在左侧菜单栏找到 Pages。

在 Branch (分支) 选项下，选择 main (或 master)，文件夹选择 /(root)。

点击 Save (保存)。

等待约 1-2 分钟，刷新页面，顶部会出现一个链接（例如 https://yourname.github.io/LayReport/ ）。

复制这个链接，发给朋友或者发朋友圈吧！💜

## ⚠️ 常见问题 Q&A
Q: 图片裂开了/不显示？

A: 请检查文件名的大小写！GitHub 是区分大小写的。Photo.JPG 和 photo.jpg 是两个不同的文件。请确保代码里的名字和实际文件名完全一致。

Q: 音乐在手机上不自动播放？

A: 现代手机浏览器的限制，必须由用户触摸屏幕后才能播放声音。模板中已经写好了逻辑，用户点击屏幕任意位置或点击右上角按钮即可播放。

Q: 可以在微信里直接打开吗？

A: 可以！通过上面步骤生成的 GitHub Pages 链接，可以直接在微信访问。
